# Исследование развития игровой индустрии с 2000 по 2013 год

### Описание проекта
Команда игры «Секреты Темнолесья» готовит статью для IT-ресурса, посвящённую развитию игровой индустрии в начале XXI века. Цель - привлечь внимание аудитории, интересующейся ретро-играми, и заинтересовать их новой игрой. Для этого необходимо исследовать исторические данные о продажах игр, платформах, жанрах и оценках пользователей и критиков.

**Задачи:**
- Загрузить и изучить данные.
- Провести предобработку (очистка, приведение типов, обработка пропусков и дубликатов).
- Отфильтровать данные за период 2000–2013 годы.
- Категоризировать игры по оценкам пользователей и критиков.
- Выделить топ‑7 платформ по количеству выпущенных игр.
- Подготовить итоговый датасет для дальнейшего анализа и визуализации.

### Выводы
- Проведена полная предобработка данных: исправлены типы, обработаны пропуски и дубликаты, нормализованы названия.
- Сформирован срез данных за 2000–2013 годы, соответствующий запросу заказчика.
- Добавлены категории оценок, что позволит в дальнейшем анализировать связь между восприятием игр пользователями/критиками и продажами.
- Выделены ключевые платформы, доминировавшие в индустрии в указанный период.

### Инструменты
