# Сравнение среднего времени активности в приложении Яндекс Книги между пользователями Санкт-Петербурга и Москвы.

### Описание проекта
В приложении «Яндекс Книги» пользователи могут читать и слушать книги. Возникла гипотеза, что аудитория из Санкт-Петербурга в среднем проводит в приложении больше времени, чем аудитория из Москвы. Для проверки этого предположения был проведён статистический анализ на основе реальных данных о сессиях пользователей.

### Цель
Проверить, является ли различие в среднем времени активности между двумя городами статистически значимым, и принять обоснованное решение о наличии или отсутствии эффекта.

### Вывод
На основе проведённого анализа нет оснований утверждать, что жители Санкт-Петербурга проводят в Яндекс Книгах статистически значимо больше времени, чем жители Москвы. Рекомендуется либо увеличить размер выборки для СПб, либо проверить гипотезу на других метриках (например, число сессий или процент дочитываний).
