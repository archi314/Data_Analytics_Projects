# Сравнение пользовательской активности в мобильном приложении для чтения книг между Москвой и Санкт-Петербургом.

### Описание проекта
В мобильном приложении для чтения и прослушивания книг требовалось проверить гипотезу о региональных различиях в поведении пользователей. Предполагалось, что аудитория из Санкт-Петербурга проводит в приложении больше времени, чем пользователи из Москвы. 

### Цель
Проверить, является ли различие в среднем времени активности между двумя городами статистически значимым, и принять обоснованное решение о наличии или отсутствии эффекта.

### Вывод
На основе проведённого анализа нет оснований утверждать, что жители Санкт-Петербурга проводят в мобильном приложении для чтения и прослушивания книг статистически значимо больше времени, чем жители Москвы. Рекомендуется либо увеличить размер выборки для СПб, либо проверить гипотезу на других метриках (например, число сессий или процент дочитываний).
