# Анализ эффективности новой версии сайта: A/B-тест по целевой метрике «конверсия в покупку».

### Описание проекта
Интернет-магазин товаров для здорового образа жизни провёл редизайн сайта, чтобы упростить интерфейс и повысить конверсию. Для оценки эффекта изменений был запущен A/B-тест.

**Гипотеза теста:** новый дизайн увеличит долю пользователей, совершивших покупку в течение 7 дней после регистрации, не менее чем на 3 процентных пункта.

### Цель исследования
- Загрузить и предобработать данные об участниках теста и событиях.
- Проверить корректность проведения теста (баланс групп, пересечения, длительность наблюдения).
- Сравнить конверсию в покупку между группами А и В с помощью статистического теста для пропорций.
- Оценить, достигнут ли целевой эффект (+3 п.п.) и является ли различие статистически значимым.
- Сформулировать выводы и рекомендации для бизнеса.

### Итоговые выводы и рекомендации
1. **Статистический эффект есть:** новый интерфейс повышает конверсию в покупку (p-value = 0,028).  
2. **Целевой бизнес-эффект не достигнут:** требуемые +3 п.п. не получены (фактически +1,73 п.п.).  
3. **Рекомендация:**  
   - Учитывая положительную динамику, можно рассмотреть частичное внедрение нового интерфейса (например, для определённых устройств или регионов).  
   - Либо доработать дизайн для достижения заявленного порога в 3 п.п. и провести повторный тест с увеличенной выборкой.
