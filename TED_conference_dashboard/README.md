# Дашборд для анализа конференций TED

## Описание проекта
Заказчик - компания, организующая первое мероприятие в духе классических конференций TED. Для успешного проведения необходимо опираться на опыт прошлых выступлений. С этой целью был разработан дашборд в DataLens, который позволяет исследовать ключевые метрики и отвечает на вопросы:

- Какие темы популярны и как зрители на них реагируют?
- Сколько в среднем выступающих на одной конференции?
- Какова оптимальная длительность выступления?
- Кто из спикеров вызывает наибольший интерес?

## Ключевые метрики и визуализации

### 1. Общая статистика
- Количество конференций
- Количество выступлений
- Уникальное количество выступающих
- Количество уникальных тегов

### 2. Детализация по конференциям
- Средняя длительность выступления
- Максимальное количество просмотров среди выступлений конференции
- Суммарное количество аплодисментов за конференцию
- Количество выступающих на конференции
- Возможность фильтрации по стране и названию конференции

### 3. Анализ выступлений
- Топ-10 самых смешных выступлений (по количеству смеха)
- Топ-20 самых популярных тегов (по числу выступлений или просмотров)

### 4. Информация о выступающих
- Самые успешные выступающие по количеству аплодисментов с указанием их рода деятельности
