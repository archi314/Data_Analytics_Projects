# Оценка нового алгоритма рекомендаций в развлекательном приложении: A/B-тест метрики вовлеченности пользователей.

### Описание проекта

Компания разрабатывает развлекательное приложение с «бесконечной» лентой коротких видео. Монетизация осуществляется через:
- **Ежемесячную подписку** - пользователь смотрит ленту без рекламы.
- **Рекламную модель** - пользователи без подписки видят рекламные вставки.

Команда рекомендательных систем создала новый алгоритм, который, по гипотезе, должен показывать более релевантный и интересный контент для каждого пользователя. Это потенциально может повысить вовлечённость (время в приложении, количество сессий, просмотры страниц), а также повлиять на метрики монетизации (конверсия в подписку, число рекламных показов).

### Цель 
Спланировать A/B-тест, проверить корректность его проведения и проанализировать результаты с помощью статистических методов.

### Вывод
Новый алгоритм не продемонстрировал статистически значимого улучшения ключевой метрики (более того, зафиксировано незначимое снижение), его внедрение в текущем виде нецелесообразно.

### Инструменты
- Python
- Pandas - загрузка, обработка и агрегация данных
- Matplotlib - визуализация данных
- Scipy.stats - статистический тест (z-test)
- **Среда выполнения**: Jupyter Notebook.
